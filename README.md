# 💰 Expense Tracker

A modern and responsive Expense Tracker web application built using HTML, CSS, and JavaScript. It helps users manage their income and expenses with an attractive interface and real-time balance calculation.

## 🚀 Live Demo

🔗 https://tanusri-dummu.github.io/Expense-Tracker/



## ✨ Features

- 💵 Add Income Transactions
- 💸 Add Expense Transactions
- 📊 Automatic Balance Calculation
- 🗑️ Delete Individual Transactions
- 🧹 Clear All Transactions
- 💾 Local Storage Support
- 🌙 Dark Mode Toggle
- 📱 Fully Responsive Design
- ✨ Smooth Animations
- 🎨 Modern Glassmorphism UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Font Awesome Icons
- Google Fonts

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Tanusri-Dummu/Expense-Tracker.git
```

2. Open the project folder

```bash
cd Expense-Tracker
```

3. Open `index.html` in your browser.

## 📖 How to Use

1. Enter a transaction description.
2. Enter the amount.
3. Click **Add Income** or **Add Expense**.
4. View your updated balance instantly.
5. Delete individual transactions or clear all data.
